# Interpretability Outputs

This folder contains Grad-CAM interpretability artifacts generated by
`interpretability_analysis.py`.

## Files

- `gradcam_samples.pdf`: multi-page PDF with Grad-CAM visualizations
- `gradcam_samples.csv`: per-sample predictions and paths
- `gradcam_*.png`: per-image Grad-CAM grids

## Regenerate

```bash
python interpretability_analysis.py
```

### Recommended (more accurate heatmaps)

```bash
# Use Score-CAM (cleaner, gradient-free) for accurate overlays
python interpretability_analysis.py --engine scorecam --cam-target true --percentile 99 --alpha 0.55 --blur 1.5 --threshold 0.35 --cmap jet
```
