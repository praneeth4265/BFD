About This Project

Bone Fracture Detection (BFD) is a production-grade deep learning system for classifying bone X-rays. It uses an ensemble of four state-of-the-art architectures ‚Äî ConvNeXt V2, EfficientNetV2-S, MaxViT-Tiny, and Swin Transformer ‚Äî combined via optimized weighted soft voting to achieve 100% accuracy on a test set of 3,082 images.

Key Highlights

  • 4-Model Ensemble: Combines CNN and Transformer architectures for robust predictions
  • Optimized Weights: Grid search over 1,771 combinations found [0.25, 0.25, 0.20, 0.30]
  • 100% Test Accuracy: Perfect classification on 540 clean test samples across 3 classes
  • Grad-CAM Heatmaps: Pinpoint fracture locations with AI explainability
  • GPU Accelerated: CUDA-powered inference for real-time predictions
  • Production API: FastAPI backend with OpenAPI documentation at /api/docs
  • 3 Classes: Comminuted fracture, Simple fracture, No fracture

Architecture

Images are preprocessed to 224√ó224, normalized with ImageNet statistics, and passed through all loaded models. Each model produces a probability distribution over the 3 classes. The ensemble combines these via weighted average (soft voting), and the class with the highest combined probability becomes the final prediction.
